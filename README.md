# VMod ModSecurity

This Vmod allows you to interface modsecurity with varnish

## How to Install
Requires: libmodsecurity3, rules

`./bootstrap; make; sudo make install`

To get more doc, man vmod_sec, or check the vcl in vcl/example.vcl
